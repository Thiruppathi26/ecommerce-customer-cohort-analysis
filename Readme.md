# E-commerce Customer Cohort Analysis Dashboard

## Project Overview

This project analyzes customer purchase behavior in an e-commerce platform using cohort analysis. The objective is to understand customer retention patterns, repeat purchase behavior, and customer lifetime value (CLV).

Cohort analysis groups customers based on their first purchase month and tracks their engagement over time to identify retention trends and revenue contribution.

## Tools & Technologies

- Power BI
- SQL / BigQuery
- Data Cleaning
- Data Modeling
- DAX
- AI-assisted query automation

## Key Metrics

- Total Revenue: $21.78M
- Customers: 5000
- Total Orders: 17K
- Average Order Value: $1.28K
- Repeat Purchase Rate: 82.16%
- Purchase Frequency: 3.41
- Customer Lifetime Value (CLV): $24.42K

## Dashboard Features

### 1. Sales Overview
- Monthly revenue trends
- Order volume analysis
- Revenue contribution by product category
- Top cities generating revenue

### 2. Customer Segmentation
- New vs Returning customers
- Customer purchase frequency
- Average order value analysis

### 3. Cohort Analysis
- Customer retention matrix
- Retention curve visualization
- Revenue contribution by acquisition cohort
- Customer acquisition trends

## Key Insights

- Customer retention drops significantly after the first purchase but stabilizes around 18–22% in later months.
- Electronics contributes the largest share of total revenue.
- A few major cities drive the majority of sales.
- Early acquisition cohorts generate the highest revenue.

## Business Recommendations

- Improve first-month retention through targeted marketing campaigns.
- Increase cross-selling strategies for high-value product categories.
- Focus customer acquisition efforts on high-performing regions.
